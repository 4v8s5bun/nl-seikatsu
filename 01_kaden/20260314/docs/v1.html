メンテナンス軽減の主要機能

自動洗剤投入 (Auto-Dose)

洗濯物の量に応じて適量の洗剤を自動投入。手動計量が不要になり、入れすぎによる残留物やカビの原因を防止。

i-DOS (Bosch) / TwinDos (Miele) / AutoDose (AEG) / UniversalDose (AEG)

セルフクリーニングコンデンサー

乾燥中に熱交換器のリント(糸くず)を自動で洗い流す。これがないと定期的な手動清掃が必要で、怠ると乾燥性能が大幅に低下。

Bosch (Serie 6+) / Siemens (iQ500+)

ヒートポンプ乾燥

低温で衣類を乾燥。衣類への負担が少なく(衣類寿命が延びる)、機械自体の熱負荷も低く、省エネ性能も大幅に高い。洗濯乾燥機では非常に稀。

該当: Miele WTI 370 WPM のみ(このリスト内)

AquaStop (漏水防止)

二重壁ホース+電子センサーで漏水を検知し自動止水。水損害を防止。Bosch/Siemens/Mieleはこのシステムに生涯保証を提供。

Bosch / Siemens / Miele / AEG (上位モデル)

インバーター / ブラシレスモーター

カーボンブラシがないため摩耗なし=モーター長寿命+静音。現行の主要メーカーはすべてインバーター搭載。LG Direct Driveモーターは10年保証付き。

全主要メーカー(現行モデルでは標準)

ドラム自動洗浄

高温プログラムでドラム内の雑菌・カビ・臭いを除去。ほとんどの機種に搭載されているが、自動リマインダー機能があるかが差。

Samsung (Self Clean) / LG (Tub Clean) / Bosch / AEG / Miele
